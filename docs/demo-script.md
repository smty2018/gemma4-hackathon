# Hackathon demo script

## Hero path: electricity bill

1. Upload a photographed electricity bill.
2. Ask: “Explain this in Bengali like you are speaking to my grandmother.”
3. Show the due amount, deadline, and charge breakdown with highlighted evidence.
4. Ask: “How much is the total after the late fee?” and run the deterministic calculator.
5. Ask: “Remind me one day before.”
6. Show the proposed reminder and require confirmation before creating it.
7. Ask: “Where can I pay in person?” and show an official office result on a map.

## Judge-facing proof points

- Same document, different language and audience style.
- Gemma performs multimodal extraction and structured tool selection.
- Deterministic code verifies sensitive facts.
- The user stays in control of every external action.
- Page-level evidence makes the answer inspectable rather than magical.
